**UPI Transactions Analysis Dashboard (Power BI)**

This project analyzes **UPI (Unified Payments Interface) transaction data** using Power BI to uncover patterns in digital payments across different cities, banks, and user segments. The dashboard provides a comprehensive view of transaction volume, user behavior, and payment trends, helping understand the growth and usage of digital payments.


* Power BI
* DAX (Data Analysis Expressions)
* Excel (Data Cleaning & Preparation)
* Data Visualization

**Dataset Description**

The dataset contains detailed UPI transaction records, including:

* **Transaction Details:** Transaction amount, type, status (Success/Failed), and payment method
* **User Demographics:** Age group, gender
* **Bank Information:** Bank names (sender & receiver)
* **Geographic Data:** Cities (Mumbai, Delhi, Bangalore, Hyderabad)
* **Device Usage:** Device type used for transactions
* **Merchant Data:** Platforms like Amazon, Flipkart, Swiggy, Zomato, IRCTC
* **Purpose of Transaction:** Categorization of payment intent

**Key Features**

  **Monthly Trend Analysis:** Tracks transaction volume across all months in 2024
  **City-wise Comparison:** Identifies top-performing cities in terms of transaction value
  **Bank Performance Analysis:** Compares transaction distribution across major banks
  **Customer Segmentation:** Analysis by age groups and gender
  **Merchant Insights:** Evaluates spending patterns across popular platforms
  **Device Usage Analysis:** Understands how users transact (mobile, etc.)
  **Transaction Status Tracking:** Success vs Failed transaction distribution
  **Dynamic Filters:** Interactive slicers for deep analysis

### **Key Insights (Interpretation)**

* **Consistent Transaction Volume:** Monthly transactions remain relatively stable (~1.6M–1.7M), indicating steady UPI usage throughout the year 
* **Top Performing Cities:** Mumbai and Delhi show the highest transaction values, closely followed by Hyderabad and Bangalore 
* **Bank Distribution:** Major banks like HDFC, SBI, Axis, and ICICI contribute almost equally, showing a competitive ecosystem 
* **High Merchant Activity:** Platforms like Amazon, Flipkart, Swiggy, and Zomato dominate transaction volumes, reflecting strong e-commerce and food delivery usage
* **Demographic Trends:** Certain age groups contribute significantly more to transaction volume, indicating higher digital adoption
* **High Success Rate:** Majority of transactions are successful, indicating reliability of UPI systems



The dashboard provides valuable insights into **digital payment behavior**, helping:

* Understand customer usage patterns
* Identify high-performing regions and platforms
* Support decision-making for fintech strategies
* Analyze growth and reliability of UPI systems
